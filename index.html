Project Description

This interface is a self-portrait built from platform traces. When JSON files are dropped in, color-coded data renders as layered backgrounds while red notification dots accumulate. Ads viewed, refused, or blocked stack as translucent overlays on an avatar, revealing how attention is captured and self-presentation is pressured into constant editing. An erase mode lets me scrub traces — yet even refusals are re-ingested as new control signals. The piece stages an agency paradox: I seem to act freely, but every action becomes another datapoint inside a designed current.
